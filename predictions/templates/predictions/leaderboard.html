How the table is ordered

The leaderboard is sorted automatically so players with stronger prediction records rise to the top.

1
Points first

Correct prediction = 3 points. Wrong or missed prediction = 0 points.

2
Then correct picks

If points are tied, the player with more correct predictions ranks higher.

3
Then missed picks

If still tied, fewer missed finished matches ranks higher, then username.
